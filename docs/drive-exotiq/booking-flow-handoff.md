# Drive Exotiq Booking Flow — Scaffold Source of Truth

Build the eight-screen mobile-web renter booking flow from the Gold + Editorial Type handoff.

Ship variants:
1. Vehicle — S1_D Dashboard
2. Dates — S_Dates
3. Driver — S_Driver
4. Extras — S_Extras
5. Protect — S2_A Cards
6. Review — S3_C Grouped
7. Pay — S_Pay
8. Confirmation — S4_E Hero Card

Design locks:
- Accent: `#C8A664`
- Button ink: `#1A1308`
- Error: `#FF4D6A`
- Warning/decline: `#FFB84D`
- Background: `#0D0F14`
- Surface: `#161922`
- Newsreader only for `.h-title` moments.
- Inter for everything else, especially prices.
- No emoji.
- Confirmation title must be car-specific: “Your McLaren is reserved.”

Product locks:
- Premium protection is default-selected.
- Decline is visually demoted and requires an explicit warning treatment.
- Two-party billing must be explicit.
- Pay screen is restrained; no urgency/trust-badge clutter.
- Current homepage remains untouched.
